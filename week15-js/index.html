Ответы на вопросы

1. Сколько есть видов условных операторов?

Ответ

1. условный оператор if (с одной ветвью)

2. условный оператор if...else (с двумя ветвями)

3. условный оператор else if... (с несколькими ветвями)

4. тернарный оператор ?:

5. оператор выбора switch

2. Перепишите пример про определение времени суток через тернарный оператор

let date = new Date(); let time = date.getHours(); if (time < 10) { alert( "Доброе утро!" ); } else { alert( "Добрый день!" ); }

Ответ

let date = new Date(); let time = date.getHours(); (time < 10) ? alert( "Доброе утро!" ) : alert( "Добрый день!" );

5. Назовите три способа назначения обработчиков событий. Какой из них самый универсальный?

Ответ

1. Назначение обработчика прямо в разметке HTML с помощью атрибута on<событие> . Только HTML.

2. С помощью свойства DOM-элемента on<событие> . Сам обработчик пишется в скрипте, HTML + JS

3. Доступ к элементу через this

Более универсальный способ назначение обработчика событий с помощью атрибута on<событие>

6. Корректна ли такая запись?

button.onclick = hello();

Ответ

Эта запись не корректна, т.к. функция должна быть присвоена как hello. Если добавить скобки, то hello() – это уже вызов функции, результат которого (равный undefined, так как функция ничего не возвращает) будет присвоен onclick.

Правильно: button.onclick = hello;

7. Какие есть события у клавиатуры?

Ответ

1. keydown возникает при нажатии клавиши клавиатуры и длится, пока нажата клавиша

2. keyup возникает при отпускании клавиши клавиатуры

3. keypress возникает при нажатии клавиши клавиатуры, но после события keydown и до события keyup. Надо учитывать, что данное событие генерируется только для тех клавиш, которые формируют вывод в виде символов, например, при печати символов. Нажатия на остальные клавиши, например, на Alt, не учитываются.

8. Что такое объект события и какие у него могут быть свойства?

Ответ

Объект события (Event) - это объект, который создает браузер, когда происходит какое-либо событие, записывает в него детали и передаёт его в качестве аргумента функции-обработчику. В объект браузер помещает много различной информации. Например, для события click: какая клавиша нажата, координаты курсора и др.

Свойства объекта Event:

event.bubbles – логическое значение, указывающее на то является ли данное событие всплывающим;

event.cancelable – определяет можно ли событие отменить;

event.clientX и event.clientY – координаты курсора относительно левого верхнего угла viewport;

event.cancelBubble – при установке true предотвращает всплытие события, т.е. оно всплывать не будет (является псевдонимом метода stopPropagation);

event.currentTarget – элемент, привязанный к обработчику события;

event.target – элемент, который создал событие;

event.timestamp – время, когда произошло событие;

event.type – тип (имя) события.

9. Самостоятельно разберитесь, какие бывают операторы сравнения? Напишите сюда как выглядят сравнение "равно", "не равно", "больше чем".

Ответ

Операторы сравнения в JavaScript:

Больше/меньше: a > b, a < b.

Больше/меньше или равно: a >= b, a <= b.

Равно: a == b. Для сравнения используется двойной знак равенства ==. Один знак равенства a = b означал бы присваивание.

Не равно: a != b.

Все операторы сравнения возвращают значение логического типа:

true – означает «да», «верно», «истина».

false – означает «нет», «неверно», «ложь».
