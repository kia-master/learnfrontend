Questions and Answers

  1. Что такое валидация? - автоматическая проверка кода по установленным правилам и шаблонам.
  2. Какой спецсимвол используется для указания знака ©? -  &copy;  подсмотрела здесь.
  3. Найдите пример еще тега HTML5, который не вошел в урок - тег  <datalist>  - стандартное текстовое поле, только при вводе слова, которое есть в списке - в выпадающем списке это слово появится. При клике на поле - покажет выпадающий список со всеми значениями. Содержит набор опций  <option> , доступных для выбора. Выбранное значение будет установлено для элемента  <input> , с атрибутом  list . Подсмотрела здесь.
  4. Какой тег мы используем для создания меню? - для создания меню используется тег  <nav> 
  5. Чем плохо использовать  div  для всего?
  6. Самостоятельно разберитесь, как вставлять в страницу видео тегами HTML5 и приложите пример кода
  7. Как сделать блок радиокнопок, чтобы выбирался только один пункт?
  8. Каким тегом можно сделать поле ввода даты?
