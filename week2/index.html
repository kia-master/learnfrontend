Questions and Answers

  1. Что такое валидация?

    Автоматическая проверка кода по установленным правилам и шаблонам.

  2. Какой спецсимвол используется для указания знака ©?

    Спецсимвол &copy;

  3. Найдите пример еще тега HTML5, который не вошел в урок - тег

    Тэг <datalist>.
    Стандартное текстовое поле. При вводе слова появляется выпадающий список - только если это слово есть в листе. При клике на поле - покажет список со всеми значениями, но можно ввести и другое. Очень удобно, можно предлагать значения уже существующие в базе данных
    Содержит набор опций <option>, доступных для выбора. Выбранное значение будет установлено для элемента <input>, с атрибутом list.
    Подсмотрела 🔗 здесь 🔗.

  4. Какой тег мы используем для создания меню?

    Для создания меню используется тег <nav>

  5. Чем плохо использовать div для всего?

    Тег div - это обычная прямоугольная область, которая не несет никакой информации. Соответсвенно для поисковых систем все элементы страницы будут одинаковыми и структура документа с тегами div менее читаема и понятна на всех этапах проектирования, от разработки до публикации.

  6. Самостоятельно разберитесь, как вставлять в страницу видео тегами HTML5 и приложите пример кода.

    В HTML5 используется тег video для добавления видео контента. Более подробно можно ознакомиться здесь. 

                                    Пример кода
    
                                <!-- Simple video example -->
                                <video 
                                src="videofile.ogg" autoplay poster="posterimage.jpg">
                                    Sorry, your browser doesn't support embedded videos,
                                    but don't worry, you can <a href="videofile.ogg">download it</a>
                                    and watch it with your favorite video player!
                                </video>
                                
                                Источник
  7. Как сделать блок радиокнопок, чтобы выбирался только один пункт?

    Использовать тег формы input с типом radio:
    <input type="radio" />

  8. Каким тегом можно сделать поле ввода даты?

    Элементы тега <input> типа date создают поля ввода и позволяют пользователю ввести дату. Код <input type="date" />

  9. Как определяется, какой именно пункт будет выбран по умолчанию в выпадающем списке?

    Определяется с помощью атрибута selected

  10. Что такое хорошая разметка?

    Простая и понятная, нет ничего лишнего, работает, проходит валидацию, соответсвует стилю написания кода в компании
