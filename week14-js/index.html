Ответы на вопросы

1. Какие есть способы объявления функций?

Ответ

1. Function Declaration (основной)

2. Function Expression

3. Стрелочные функции (самый новый)

2. Приведите примеры вызова одной и той же функции всеми известными вам способами.

Ответ

Function Declaration

function sum(a, b) { return a + b; }

Function Expression

var sum = function(a, b) { return a + b; };

Arrow functions

let sum = (a, b) => a + b;

3. В чем разница между тестированием и отладкой (дебаггингом)? А что такое логирование?

Ответ

Когда проводят тестирование - тестируют продукт (сайт) - выявляют ошибки. Когда проводят отладку, исправляют ошибки, которые были выявлены в ходе тестирования.

Логирование - это также отладочный инструмент, который позволяет вывести необходимую информацию в консоль или файл.

4. В чем разница между Function Expression и Function Declaration?

Ответ

1. В объявлении

2. С помощью Function Expression результаты функции можно сразу присвоить переменной.

3. Function Expression создаётся, когда выполнение доходит до него, и затем уже может использоваться.
Function Declaration можно использовать во всем скрипте (или блоке кода, если функция объявлена в блоке).

5. Что делает функция console.log() ?

Ответ

Для вывод информации в консоль. Например, console.log(text), выведет в консоль содержимое переменной text

6. Что такое BOM и DOM?

Ответ

Document Object Model, сокращённо DOM – объектная модель документа, которая представляет все содержимое страницы в виде объектов, которые можно менять. В модели DOM каждый HTML-тег является объектом. Вложенные теги являются «детьми» родительского элемента. Текст, который находится внутри тега, также является объектом.

Browser Object Model, BOM (Объектная модель браузера) – это дополнительные объекты, предоставляемые браузером (окружением), чтобы работать со всем, кроме документа, например, информация о самом браузере и операционной системе, получение текущего URL и т.п.

7. Есть вот такая страница:

Как найти в ней?…

1. Таблицу с id="age-table"

let table = document.getElementById('age-table');

2. Все элементы label внутри этой таблицы (их три)

table.getElementsByTagName('label'); // или let label = document.querySelectorAll('#age-table label');

3. Форму form с именем name="search"

let form = document.querySelector('form[name="search"]');

8. Как сделать переход на другую страницу при клике на кнопку (без <a href=...>, только средствами JavaScript)?

Ответ

С помощью BOM-объекта location :

<button onclick="window.location='https://www.google.com'"> Click Me </button>

9. Как можно обнулить (очистить) значение внутри input?

Ответ

let input = form.getElementsByTagName('input'); input.value = ''; // или input.reset();

10. Как будет выглядеть ваша функция приветствия из прошлого домашнего задания, если ее переписать в стрелочном формате?

Function Declaration

function getName() { let name = prompt('Как тебя зовут?'); alert(`Добро пожаловать ${name}!`); }

Arrow function

let getName = () => alert(`Добро пожаловать ${prompt('Как тебя зовут?')}!`);
