Task 5

  1. Kotэ: Что такое атрибут тега? Какие есть атрибуты у тега img?
    Ирина: Атрибут тега - это свойство тега, которое позволяет форматировать текст, изменять картинку. У тэга img есть следующие атрибуты:
    • src - ссылка на картинку
    • alt - описание картинки
  2. Kotэ: Зачем нужен тег span?
    Ирина: Тег span нужен для выделения блока текста без переноса строки. Необходим для форматирования одного или несколько слов.
  3. Kotэ: Как добавить иконку? Ирина: Для добавления иконки используется тэг link с атрибутами:
    • rel - указываем короткое имя
    • href - ссылка на иконку
    • type - тип image/svg или png, в зависимости от типа иконки
  4. Kotэ: Как сделать маркированный список?
    Ирина: Список помещаем в тег ul, а каждый подпункт списка в теги li.
  5. Kotэ: А как выделить слово жирным шрифтом?
    Ирина: Можно внутри текста выделить тегом b, а можно внутри стилевых тегов с использованием атрибута style и свойства внутри font-weight:bold.
